В данном задании необходимо доработать скрипт в файле app.py, добавив условный оператор. В итоге получится программа, которая позволяет определить, на какие значения делится число без остатка.

Скачайте файл app.py из закрепленных файлов и вместо комментария "# Вместо этого комментария напишите решение" впишите решение следующей задачи: 

Реализуйте условный оператор, который определяет кратность числа в переменной value

1) Если число value кратно двум, необходимо создать переменную result со значением "variable is a multiple of two"
2) Если число value кратно трем, необходимо создать переменную result со значением "
variable is a multiple of three"
3) Если число value кратно пяти, необходимо создать переменную result со значением "variable is a multiple of five".
4) В ином случае создать переменную result со значением "variable is not a multiple of 2, 3 and 5"

После решения задачи необходимо скачать файл submit.py из закрепленных файлов и запустить его в той же директории, где расположена программа app.py с решением задачи. 

Обратите внимание: для корректной работы файла submit.py необходимо скачать пакет requests. Для этого в терминале необходимо выполнить следующую команду:

Для Windows 
pip install requests

Для Mac/Unix
pip3 install requests

В данном задании необходимо доработать скрипт в файле app.py, добавив цикл. В итоге должна получиться программа, которая позволяет посчитать сумму чисел в указанном диапазоне.

Скачайте файл app.py из закрепленных файлов и вместо комментария "# Вместо этого комментария напишите решение" впишите решение следующей задачи:

Реализуйте цикл, который определяет сумму простых чисел, входящих в диапазон от 0 до значения переменной value.

Значение суммы простых чисел необходимо присвоить переменной result.

После решения задачи необходимо скачать файл submit.py из закрепленных файлов и запустить его в той же директории, где расположена программа app.py с решением задачи.

Обратите внимание: для корректной работы файла submit.py необходимо скачать пакет requests. Для этого в терминале необходимо выполнить следующую команду:

Для Windows 
pip install requests

Для Mac/Unix
pip3 install requests
